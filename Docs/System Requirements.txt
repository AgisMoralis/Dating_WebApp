Install Angular 18.2 by executing the following commands on terminal (through VSCode):
- "npm install -g @angular/cli@18.2" (installs the angular 18.2)
- "ng version" (checks if the installation was successful)

Download the Installer from the official GitHub page and install nvm:
- "https://github.com/coreybutler/nvm-windows/releases"
Execute the following command on terminal (through VSCode):
- "nvm version" (checks if the installation was successful)

Install the Node.js (v22.14.0) by executing the following commands on terminal (through VSCode):
- "nvm install 22.14.0" (install Node.js version 22.14.0)
- "nvm use 22.14.0" (defines that the current Node.js version shall be used)
